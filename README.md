# Essential 80s Metal Playlist

A cinematic, login-protected 80s metal and hard rock playlist experience built with pure HTML, CSS, and JavaScript 😎.

This project showcases legendary glam metal, hard rock, and heavy metal artists through immersive full-screen sections, custom audio playback and animated UI design 🥂

---

# Features

- Full-screen cinematic band sections
- Smooth scrolling navigation
- Embedded audio playback system
- Firebase email/password authentication (Removed for static purposes)
- Hosted on Github Pages
- EULA-protected licensed access
- Anti-copy protections
- Watermark overlay system (Too lazy to finish it 😎)
- Responsive mobile design
- Custom glam metal visual styling
