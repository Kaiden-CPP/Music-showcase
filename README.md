# Essential 80s Metal Playlist

A cinematic, login-protected 80s metal and hard rock playlist experience built with pure HTML, CSS, and JavaScript.

This project showcases legendary glam metal, hard rock, and heavy metal artists through immersive full-screen sections, custom audio playback, animated UI design, and Firebase-based account authentication.

---

# Features

- Full-screen cinematic band sections
- Smooth scrolling navigation
- Embedded audio playback system
- Firebase email/password authentication
- GitHub Pages compatible
- EULA-protected licensed access
- Anti-copy protections
- Watermark overlay system
- Responsive mobile design
- Custom glam metal visual styling
